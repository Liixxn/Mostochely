<h1 align="center">🍕Mostochely🍕</h1>
<hr>
<p>Mostochely es una aplicación de escritorio que permite la gestión de una pizzería, en la cúal se puede realizar un pedido, creando su pizza con su respectiva masa e ingrediente. Además de enfocarse en el usuario, también se encuentra dirigida al personal de este esteblacimiento, en concreto el repartirdor. Éste podrá confirmar la llegada de un pedido como cancelarlo.</p>

<p>Este proyecto fue desarrollado para la asignatura "Fundamentos de Programción" de 1º año de la carrera de Ingeniería Informática. Se utilizó Java, Java Swing para la interfaz, y se el estilo de arquitectura MVC (Modelo-Vista-Controlador).</p>

<h2 align="center">Información sobre archivos de interés</h2>

> Se van a explicar la funcionalidad de algunos archivos para entender mejor cómo funciona la aplicación:
<ul>
  <li>📕<i>miglayout15-swing.jar</i>: Es una librería que ayuda a la distribución de los elementos a la hora de diseñar la interfaz. Se debe añadir al "Build Path" del proyecto. </li>
  <li>📑<i>Archivos .txt</i>: Los archivos con extensión .txt son indespensables para el manejo de la pizzería. En ellos se almacena la información de las pizzas y los pedidos.</li>
  <li>🍕<i>Principal.java</i>: Es el fichero principal que pone en funcionamiento la apliación</li>
  <li>🖥<i>index.html</i>: Este archivo se encuentra dentro de la carpeta "doc", al hacer doble click sobre él, se abre la documentación del proyecto dónde se puede adquirir breves descripciones de lo que hace cada función.</li>
</ul>
